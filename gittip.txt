1. 폴더를 하나 만든다 
2. git init로 메인 작업 폴더를 설정한다.
3. git clone 사용자명 @ 호스트blabla 깃허브 웹상에서 만든 repositrory
와 연결을 한다.
4. git status 로 상태화인 
5. 작업 진행 ..
6. 만든 파일을 git add 파일명으로 폴더에서 중간 연결 버스로 옮긴다.
7. git commit -m "부가설명" 명령으로 올라갈 녀석들에 대한 설명을 추가한다.
8. git pull 로 원격 저장소와 로컬저장소 동기화를 한번해준다.
9. git push origin 가지이름(대부분 master) 으로 원격 저장소로 올린다
